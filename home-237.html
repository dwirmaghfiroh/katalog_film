Mata Kuliah: Pemrograman Web 1

Mata kuliah Pemrograman Web 1 merupakan wadah bagi mahasiswa untuk mempelajari fondasi dasar dalam pengembangan aplikasi berbasis web. Fokus utama dari pembelajaran ini mencakup pemahaman struktur dokumen menggunakan HTML, implementasi estetika serta tata letak responsif menggunakan CSS, hingga pemberian logika interaktif pada sisi klien (client-side scripting) menggunakan JavaScript. Melalui mata kuliah ini, mahasiswa diharapkan mampu merencanakan, menganalisis, serta menciptakan perangkat lunak berbasis web yang fungsional dan tepat guna.
