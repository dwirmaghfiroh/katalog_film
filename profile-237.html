Tentang WEB

Website Katalog Film ini dikembangkan khusus sebagai implementasi tugas akhir (UAS) mata kuliah Pemrograman Web 1. Situs ini berfungsi untuk menyajikan informasi perfilman terkini sekaligus memberikan simulasi fitur interaktif berupa sistem pemesanan/pembelian tiket nonton secara mandiri.


Fitur utama yang diimplementasikan pada sistem ini meliputi:

  • Layouting modern bernuansa bioskop digital menggunakan CSS murni.
  • Halaman Transaksi interaktif berbasis formulir input tiket film.
  • Pengolahan data dinamis (Create & Read) ke dalam bentuk tabel menggunakan JavaScript.
  • Validasi keamanan form login di sisi klien (Client-side Validation).
